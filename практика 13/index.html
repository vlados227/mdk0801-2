Массивы в JavaScript

Массивы — это особый тип объекта в JavaScript, который позволяет хранить несколько значений в одной переменной. Каждый элемент массива имеет свой индекс, начиная с нуля. Массивы могут содержать элементы любого типа: числа, строки, другие массивы и даже объекты.

Основные методы массивов

  • push() — добавляет элемент в конец массива.
  • pop() — удаляет последний элемент массива.
  • shift() — удаляет первый элемент массива.
  • unshift() — добавляет элемент в начало массива.
  • length — возвращает количество элементов в массиве.
  • forEach() — выполняет указанную функцию для каждого элемента массива.

Дополнительные методы работы с массивами

  • map() — создает новый массив, результатом которого является применение функции к каждому элементу исходного массива. 
    let arr = [1, 2, 3];
let doubled = arr.map(x => x * 2); // [2, 4, 6]
  • filter() — создает новый массив, содержащий элементы, которые удовлетворяют условию функции. 
    let arr = [1, 2, 3, 4, 5];
let even = arr.filter(x => x % 2 === 0); // [2, 4]
  • reduce() — выполняет функцию "свертки", применяя её к каждому элементу массива для вычисления одного итогового значения. 
    let arr = [1, 2, 3, 4];
let sum = arr.reduce((acc, x) => acc + x, 0); // 10
  • concat() — объединяет два или более массивов. 
    let arr1 = [1, 2];
let arr2 = [3, 4];
let combined = arr1.concat(arr2); // [1, 2, 3, 4]
  • includes() — проверяет, содержит ли массив определённое значение. 
    let arr = [1, 2, 3];
let hasTwo = arr.includes(2); // true

Многомерные массивы

В JavaScript можно создавать массивы, содержащие другие массивы. Такие массивы называются многомерными.

let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

// Доступ к элементу
console.log(matrix[1][2]); // 6

Работа с массивами объектов

Массивы могут содержать объекты. Это полезно при работе с данными, например, в списке пользователей.

let users = [
    { name: "John", age: 30 },
    { name: "Jane", age: 25 }
];

// Доступ к свойствам объектов в массиве
console.log(users[0].name); // "John"
